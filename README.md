# Laboratório de Experimentação e Medição de Software

Repositório com os laboratórios práticos desenvolvidos para a disciplina de **Laboratório de Experimentação e Medição de Software** da **PUC Minas**.

O objetivo geral da disciplina é aplicar métodos empíricos para analisar características de repositórios de software open-source, utilizando dados coletados via API do GitHub e ferramentas de análise estática de código.

## Laboratórios

| Lab | Descrição |
|-----|-----------|
| [`lab01/`](lab01/) | Análise de repositórios populares do GitHub — coleta de métricas de processo (estrelas, PRs, issues, releases) e análise estatística |
| [`lab02/`](lab02/) | Análise de qualidade interna de repositórios Java — correlação entre métricas de processo e métricas de produto (CBO, DIT, LCOM) calculadas via ferramenta CK |

Cada laboratório possui seu próprio `README.md` com instruções detalhadas de instalação e execução.
